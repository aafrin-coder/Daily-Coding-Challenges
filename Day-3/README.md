# Day 03

## Problem 1

**Name:** Palindrome Number

- Platform: LeetCode
- Difficulty: Easy
- Language: Java
- Approach: Reversed half of the number and compared it with the remaining half to determine whether the integer is a palindrome without converting it to a string.
- Time Complexity: O(log n)
- Space Complexity: O(1)

## Problem 2

**Name:** Delete Duplicate Emails

- Platform: LeetCode
- Difficulty: Easy
- Language: MySQL
- Approach: Used a self JOIN on the `Person` table to identify duplicate email addresses and deleted the duplicate records while keeping the record with the smallest `id`.
- Time Complexity: O(n²)
- Space Complexity: O(1)
