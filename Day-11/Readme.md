# Day 11

## Problem 1

**Name:** Missing Number

- Platform: LeetCode
- Difficulty: Easy
- Language: Java
- Approach: Calculate the expected sum of numbers from `0` to `n` while simultaneously calculating the sum of the given array. The missing number is the difference between the expected sum and the actual sum.
- Time Complexity: O(n)
- Space Complexity: O(1)

## Problem 2

**Name:** Move Zeroes

- Platform: LeetCode
- Difficulty: Easy
- Language: Java
- Approach: Use a pointer to keep track of the position where the next non-zero element should be placed. Traverse the array, move all non-zero elements to the front, and fill the remaining positions with zeroes.
- Time Complexity: O(n)
- Space Complexity: O(1)
