# Day 04

## Problem 1

**Name:** Customer Placing the Largest Number of Orders

- Platform: LeetCode
- Difficulty: Easy
- Language: MySQL
- Approach: Grouped orders by `customer_number`, counted the number of orders for each customer, sorted the results in descending order, and returned the customer with the highest order count.
- Time Complexity: O(n)
- Space Complexity: O(1)

## Problem 2

**Name:** Excel Sheet Column Number

- Platform: LeetCode
- Difficulty: Easy
- Language: Java
- Approach: Treated the column title as a base-26 number. Iterated through each character, converted it to its corresponding numeric value, and accumulated the result.
- Time Complexity: O(n)
- Space Complexity: O(1)
