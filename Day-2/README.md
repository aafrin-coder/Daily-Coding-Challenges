# Day 02

## Problem 1

**Name:** Combine Two Tables

- Platform: LeetCode
- Difficulty: Easy
- Language: MySQL
- Approach: Used a LEFT JOIN to combine the `Person` and `Address` tables so that all persons are included, even if they do not have an address.
- Time Complexity: O(n)
- Space Complexity: O(1)

## Problem 2

**Name:** Employees Earning More Than Their Managers

- Platform: LeetCode
- Difficulty: Easy
- Language: MySQL
- Approach: Performed a self JOIN on the `Employee` table to compare each employee's salary with their manager's salary and selected employees earning more than their managers.
- Time Complexity: O(n)
- Space Complexity: O(1)
