# Day 06

## Problem 1

**Name:** Binary Search

- Platform: LeetCode
- Difficulty: Easy
- Language: Java
- Approach: Apply Binary Search by maintaining two pointers (`left` and `right`). Repeatedly calculate the middle index, compare the target with the middle element, and search the appropriate half until the target is found or the search space becomes empty.
- Time Complexity: O(log n)
- Space Complexity: O(1)

## Problem 2

**Name:** Rising Temperature

- Platform: LeetCode
- Difficulty: Easy
- Language: MySQL
- Approach: Use the `EXISTS` clause to compare each day's temperature with the previous day's record. Return the IDs of days where the current temperature is higher than the previous day's temperature.
- Time Complexity: O(n)
- Space Complexity: O(1)
